# Fixing Python SSL Certificate Failed Error

import os, ssl

if (not os.environ.get('PYTHONHTTPSVERIFY', '') and
getattr(ssl, '_create_unverified_context', None)):
    ssl._create_default_https_context = ssl._create_unverified_context

Scraping DJIA From Wikipedia¶

"The DJIA includes only 30 large companies and is a price-weighted index, unlike later stock indices which use market capitalization." -Wikipedia

Price-Weighted Index

Capitalization-Weighted Index

In this section we're going to grab all 30 tickers and append it to a list in order to grab historical data using yfinance later on. To keep things simple, instead of using beautiful soup, I decided to just use the built in pandas .read_html() method that is able to fetch data from tables. More info. about this method can be found in the official docs here:

Unicode Normalization¶

Sometimes when using "pd.read_html( )" the scraped info. from tables contains unicode strings that cannot be seen when using the .info() method (it will just say the data types within the column are "objects"). We have to manually check ourselves by looking at the data - so in the example above we manually check the columns for unicode which would look something like (if looking at unicode data in the "Symbols" column):

"NYSE:\xa0MMM"
"NYSE:\xa0AXP"

Note: We see that we do not have any unicode strings in our scraped table from wikipedia, but this could be due to the recent jupyter notebooks & tools version I have when doing this project. Below is an image of all the current versions I'm using.

Backtesting a Simple Momentum Strategy¶

Before we start let's go over some financial terms and what our strategy entails.

Long Position: Purchase of an asset with the expectation that it will increase in value over time.

Short Position: When an investor anticipates the price of a security will fall in the short term (opposite of a long position).

Note: While you cannot invest directly in the DJI index itself (you can definitely still invest in each of the companies that make up the index), there are exchange traded funds that yield similar returns to the performance of DJI. One such ETF is the SPDR Dow Jones Industrial Average ETF Trust (DIA)

For simplicity's sake, in this project we are going to assume you can invest directly in DJI. And we will use the following metrics to gauge our positions:

+1: DJI Long position

-1: DJI Short position

0: DJI No/ Neutral position

We're going to backtest our active (simple momentum) strategy against just buying and holding (passive)

  1. Buying and Holding:

    • Initially investing into DJI and do nothing (Position: +1 on any given day)

  1. Simple Momentum:

    • At the end of the day if today's return was positive, then we will invest and take a long position:

      • +1 into DJI if today's return was positive

    • At the end of the day if today's return was negative, then we will short sell the DJI and take a short position

      • -1 into DJI if today's return was negative
